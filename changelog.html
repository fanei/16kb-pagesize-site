Version history and updates for 16KB Pagesize resources and optimizations.

v0.8.8 (ijkplayer SO)
2024-12-12

Optimizations:

  • Added size-optimized builds for all Android ABIs (arm64-v8a, armeabi-v7a, x86, x86_64)
  • Enabled Link-Time Optimization (LTO) for smaller binary size
  • Stripped debug symbols and unused code sections
  • Reduced .so file size by ~40% compared to standard builds
  • Provided SHA256 checksums for all architecture variants

Build Details:

  • Toolchain: NDK r21e, Clang 12.0
  • Optimization flags: -Os -flto -fvisibility=hidden
  • Average file size: ~14KB per ABI (uncompressed), ~8KB (compressed)
